### 배열 메서드 

### splice
> 주어진 배열의 원하는 위치에서 요소를 제거, 추가, 교체할 수 있는 메서드
> 삭제된 요소들을 반환
> 원본 배열 변경됨

splice 매개변수 
1. 시작할 인덱스 위치
2. 삭제할 요소 갯수
3. 추가할 요소

```js
arr.splice(index,[deleteCount, element1, ...]);
```


### slice
> 배열의 일부분을 잘라서 새로운 배열을 복사하는 메서드
> 새로운 배열을 반환

```js
arr.slice([index], [end])
```
slice 매개변수
1. 복사를 시작할 인덱스 
2. 복사를 끝낼 인덱스 직전까지
