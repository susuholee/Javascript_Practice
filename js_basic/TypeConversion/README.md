### 형 변환(Type Conversion)
> 데이터의 타입을 다른 타입으로 변환하는 것을 뜻함
> 자바스크립트는 타입 검사가 유연한편

### 형 변환 목적
> 다른 타입의 데이터들의 연산이나 비교, 조건식을 처리해야할 때 사용된다

### 형 변환 종류
> 1. 암시적 형 변환 (Implicit Type Conversion)
> 2. 명시적 형 변환 (Explicit Type Conversion)

## 암시적 형 변환 (Implicit Type Conversion)
> 자바스크립트 엔진이 연산이나 계산처리가 필요할때 따라서 자동으로 타입을 변환하는 것을 뜻한다

### 문자열 변환
> 문자열이 포함된 연산에서 문자열로 변환
> 모든 사칙연산 (+,-,*,/)이 문자열로 변환되는것이 아님
> 여기서 +만 문자열로 변환된다 왜?
> 그 이유는 + 연산자는 나중에 설명할 이야기지만 자바스크립트에서 + 연산자는 덧셈 연산자이자 문자열 연결 연산자로 동작되기 때문 -> 자세한 설명은 코드로 다시 설명
> -,*,/ 연산은 숫자형으로 변환 
> 그럼 나머지 연산자는 문자열로 변환되지 않을까? 
> 그 이유는 산술 연산으로만 사용되기 때문

### 숫자형 변환
> -,*,/ 연산자들은 숫자형으로 변환

### 불리언 변환 
> true or false로 변환 

### 명시적 형 변환 
> 암시적과는 달리 개발자가 의도적으로 데이터 타입을 변환
> 보통 일반적으로 형 변환하는 함수를 사용하여 변환


### 숫자형 변환
> Number Type 즉, 숫자형 타입으로 변환
1. Number()

2. parseInt()
3. parseFloat()

### 문자열 변환
> String Type 즉, 문자열 타입으로 변환
1. String()
2. value.toString()
3. toFixed()

---

### 불리언 변환
> Boolean Type 즉, 불리언 타입으로 변환
1. Boolean()
