### 비교 연산자
> 어떠한 값을 비교할때 사용되는 연산자
> 반환 타입은 불리언 타입, true or false

## 비교 연산자 종류
```js
1. > : 크다
2. < : 작다
3. >= : 크거나 같다
4. <= : 작거나 같다
5. == : 값이 같다
6. === : 값, 타입 모두 같다
7. != : 값이 다르다
8. !== : 값, 타입 모두 다르다
```

```js
let a = 200;
let b = 300;
console.log(a > b);
console.log(a < b);
console.log(a >= b);
console.log(a <= b);
console.log(a == b);
console.log(a === b);
console.log(a != b);
console.log(a !== b);
```
